The tutorials are based on a machine learning lecture by Mr. Krish Naik.
The lecture covers the following topics:
  a) Linear Regression
  b) Mean Squared Error, Mean Absolute Error, and RMSE
  c) Ridge and Lasso Regression
  d) Elastic Net Regression
  e) Logistic Regression
  f) Naive-Baye
  g) Overfitting, Underfitting, Bias, and Variance
  h) Performance Metrics, Accuracy, Precision, Recall, and F-Beta Score
  i) Support Vector Regression
  j) Decision Tree
  k) Cross-Validation
  l) Bagging and Boosting
  m) Random Forest Regression and Classification
  n) Out-of-Bag (OOB) Evaluation and Error in Random Forest
  o) Automated EDA using Pandas Profiling
  p) Automated EDA using Autoviz library
  q) In-depth EDA using Sweetviz Library
  r) R-squared and Adjusted R-squared 
  s) Boosting technique
  t) Adaboost ML
  u) Gradient Boosting Algorithms
  v) Complete Unsupervised Learning
  w) Complete Anomaly Detection
